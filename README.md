# NewspaperWeb WordPress Theme

A modern and responsive WordPress theme designed for news websites and blogs. Built with Bootstrap 5 and following WordPress best practices.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Multiple Homepage Layouts**: Choose between two different homepage designs
- **Custom Widget Areas**:
  - Main Sidebar
  - Home Sidebar
  - Header Ad Space
  - Footer Columns (2)
- **Theme Options**:
  - Custom colors
  - Typography settings
  - Layout options
  - Post display settings
- **Bootstrap Integration**: Built with Bootstrap 5 for modern UI components
- **Customizable Menus**:
  - Primary Menu
  - Footer Menu
  - Social Menu
- **Post Features**:
  - Featured images support
  - Custom post meta boxes
  - Sidebar toggle per post/page
  - Title visibility control

## Installation

1. Download the theme
2. Upload it to your WordPress themes directory (`wp-content/themes/`)
3. Activate the theme through the WordPress admin panel

## Theme Options

Access theme options through WordPress admin panel:
1. Go to Appearance > Theme Options
2. Configure:
   - Colors
   - Typography
   - Layout settings
   - Post display options

## Widget Areas

The theme includes several widget areas:
- **Sidebar**: Main sidebar for all pages
- **Home Sidebar**: Special sidebar for homepage layouts
- **Header Ad Space**: Area for advertisements
- **Footer Columns**: Two widget areas in the footer
